Overview

DreamCobots is an open-source, self-sustaining AI bot ecosystem that gives every user access to 109 top AI models — free or via tokens and subscriptions. The platform automates testing, deployment, dependency management, and cost attribution so it runs without a dedicated operations team.

Key highlights:

  • 🆓 Free tier — access all 109 models with daily token allowance
  • 💳 Token credits — pay-as-you-go for advanced usage
  • 📅 Monthly & annual subscriptions with volume discounts
  • 🤖 BuddyBot orchestrates all bots, tracks consumption, manages subs
  • 🔄 GitHub Actions CI/CD + Dependabot keep everything up-to-date
  • 🛡️ MIT licensed & community-governed

Installation

# 1. Clone the repository
git clone https://github.com/ireanjordan24/Dreamcobots.git
cd Dreamcobots

# 2. Install Python dependencies
pip install -r requirements.txt

# 3. (Optional) Verify all bots comply with the framework
python tools/check_bot_framework.py

# 4. Run the test suite
python -m pytest tests/ -q

Requirements

Python 3.8+ · pytest ≥ 7.0 · See requirements.txt for the full list.

Usage

Quick-start: BuddyBot

from BuddyAI import BuddyBot
from bots.token_billing import BillingSystem, Tier

# Create the billing system and a FREE user
billing = BillingSystem()
user_id = billing.create_account("alice", Tier.FREE)

# Create and register a bot
from bots.ai_chatbot.chatbot import Chatbot
bot = Chatbot()
buddy = BuddyBot()
buddy.register_bot("chatbot", bot)

# Route a message
response = buddy.route_message("chatbot", "Hello!")
print(response)

Using Tokens

# Purchase token credits
billing.purchase_tokens(user_id, amount=100)   # 100 tokens

# Deduct tokens for advanced usage
billing.deduct_tokens(user_id, cost=5, description="GPT-4 call")

# Check balance
balance = billing.get_balance(user_id)
print(f"Remaining tokens: {balance['tokens']}")

Subscriptions

# Upgrade to monthly Pro subscription
billing.create_subscription(user_id, Tier.PRO_MONTHLY)

# Or choose annual (save ~17%)
billing.create_subscription(user_id, Tier.PRO_ANNUAL)

FAQ

Is the free tier really free forever?

Yes. The free tier never expires. You receive a daily token allowance that refreshes every 24 hours, giving you access to all 109 AI models at limited capacity.

What happens if I run out of tokens?

Your daily allowance will refresh the next day. You can also purchase a token credit pack at any time from the billing portal for instant top-up.

Do tokens expire?

Purchased token credits never expire. Daily free tokens that are not used do not roll over.

How do I upgrade from Free to Pro?

Call billing.create_subscription(user_id, Tier.PRO_MONTHLY) or visit the billing portal. Annual plans save approximately 17%.

Can I self-host DreamCobots?

Yes — DreamCobots is MIT licensed. Clone the repo, install dependencies, and run the bots locally. You are responsible for your own API keys and infrastructure costs when self-hosting.

How does BuddyBot manage costs?

BuddyBot tracks every token deduction per user via track_usage(). All AI model costs are passed directly to the client's token balance, keeping the platform cost-neutral.

How do I report a bug or security issue?

Open a GitHub Issue for bugs. For security vulnerabilities, please email dreamcobots-security@example.com privately.
