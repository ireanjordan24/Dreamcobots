Overview

DreamCobots is an open-source, self-sustaining AI bot ecosystem that gives every user access to 109 top AI models — free or via tokens and subscriptions. The platform automates testing, deployment, dependency management, and cost attribution so it runs without a dedicated operations team.

  • 🆓 Free tier — access all 109 models with daily token allowance
  • 💳 Token credits — pay-as-you-go for advanced usage
  • 📅 Monthly & annual subscriptions with volume discounts
  • 🤖 BuddyBot orchestrates all bots, tracks consumption, manages subs
  • 🔄 GitHub Actions CI/CD + Dependabot keep everything up-to-date
  • 🛡️ MIT licensed & community-governed

Installation

# 1. Clone the repository
git clone https://github.com/ireanjordan24/Dreamcobots.git
cd Dreamcobots

# 2. Install Python dependencies
pip install -r requirements.txt

# 3. Verify all bots comply with the framework
python tools/check_bot_framework.py

# 4. Run the full test suite
python -m pytest tests/ -q

Python 3.8+ · pytest ≥ 7.0 · See requirements.txt for the full list.

Contributing

We welcome pull requests! Read CONTRIBUTING.md before submitting.

  • All bots must implement the GLOBAL AI SOURCES FLOW pipeline
  • Add tests in tests/test_<bot_name>.py
  • Run python tools/check_bot_framework.py before opening a PR
  • Run python -m pytest tests/ -q — all tests must pass
