🚀 Deploy Guide

DreamCo Main Dashboard (Flask)

git clone https://github.com/DreamCo-Technologies/Dreamcobots.git
cd Dreamcobots
pip install -r dashboard/requirements.txt
python dashboard/app.py
# → http://localhost:5001

Web Control Dashboard (Flask)

pip install -r dashboards/web_control/requirements.txt
python dashboards/web_control/run.py
# → http://localhost:5050

Deploy to Heroku

# DreamCo Main Dashboard
heroku create dreamco-main-dashboard
git subtree push --prefix dashboard heroku main

# Web Control Dashboard
heroku create dreamco-web-control
git subtree push --prefix dashboards/web_control heroku main

Enable GitHub Pages

Go to Repository Settings → Pages → Source and select main branch, docs/ folder. This page will then be live at: 

https://dreamco-technologies.github.io/Dreamcobots/dashboards.html
