# Stripe Publishable Key

The Stripe publishable key (safe to use on the client side) must be stored
in the environment or a secrets manager — never hard-coded in source files.

Configure it as a GitHub Actions secret named `STRIPE_PUBLISHABLE_KEY` and
reference it in your application via:

```
STRIPE_PUBLISHABLE_KEY=pk_live_...   # production
STRIPE_PUBLISHABLE_KEY=pk_test_...   # test / staging
```

See docs/stripe.md for full setup instructions.
