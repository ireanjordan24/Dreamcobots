# DreamCo Approved AI Tools

## Tier 1 — Fully Approved (Internal & Customer Data Permitted)
These tools have passed DreamCo's security review and may be used with internal data
under normal data-handling protocols.

| Tool | Use Case | Owner |
|------|----------|-------|
| GitHub Copilot | Code generation & review | Engineering |
| OpenAI API (GPT-4+) | Bot intelligence, content, analysis | All teams |
| Anthropic Claude API | Reasoning, summarization, research | All teams |
| Google Gemini API | Multimodal analysis, search | All teams |
| DreamCo Bots (this repo) | All automation workflows | All teams |
| Grafana | Metrics visualization | Infrastructure |
| Prometheus | Metrics collection | Infrastructure |
| Docker | Bot containerization | DevOps |
| GitHub Actions | CI/CD automation | DevOps |
| Cloudflare | Edge networking & security | Infrastructure |
| Supabase | Database & auth backend | Engineering |
| n8n | No-code workflow automation | Operations |
| LangChain | LLM orchestration | AI Engineering |

## Tier 2 — Public / Experimental Tools (Public Data Only)
These tools are permitted for use with **public, non-proprietary data only**.
Do not enter DreamCo business data, customer data, or credentials.

| Tool | Permitted Use |
|------|---------------|
| ChatGPT web interface | Research, brainstorming with public info |
| Public LLM playgrounds | Prototyping prompt templates |
| Experimental APIs (beta) | Internal PoC only, no production data |
| Unverified automation tools | Sandbox/staging only |

## Approval Process
To request a new tool be added to Tier 1, submit a request via GitHub Issues with
the label `tool-approval-request`. The Security Lead will review within 5 business days.
