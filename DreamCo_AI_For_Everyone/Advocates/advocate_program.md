# DreamCo AI Advocate Program

## What Is a DreamCo Advocate?
Advocates are team members and contributors who champion AI adoption within DreamCo.
They are the bridge between technology and people — teaching, testing, supporting, and inspiring.

---

## Advocate Responsibilities
1. **Teach**: Run workshops and demos for team members (see [Training](advocate_training.md))
2. **Test**: Beta-test new bots and provide feedback via GitHub Issues
3. **Report**: Document bugs, wins, and workflow improvements
4. **Share**: Publish workflows and use cases to GitHub Discussions
5. **Onboard**: Guide new contributors through the [Beginner Path](../Learning/beginner_path.md)
6. **Govern**: Uphold AI policies and flag violations

---

## Advocate Tiers

| Level | Requirements | Perks |
|-------|-------------|-------|
| **Starter Advocate** | Complete beginner path, run 1 bot | Community recognition |
| **Core Advocate** | 5 contributions, run 5 bots | PRO tier access, early features |
| **Lead Advocate** | 20 contributions, host 1 workshop | ENTERPRISE access, co-author credits |
| **DRI Advocate** | Own a system area | Leadership role, governance access |

---

## How to Become an Advocate
1. Complete the [Beginner Learning Path](../Learning/beginner_path.md)
2. Run at least 1 GitHub Actions workflow successfully
3. Open a GitHub Issue titled "Advocate Application: [Your Name]"
4. Tag it with the label `advocate-application`
5. Introduce yourself in [GitHub Discussions](https://github.com/DreamCo-Technologies/Dreamcobots/discussions)

---

## Advocate Network Size Goals
- FREE tier: up to 3 advocates
- PRO tier: up to 25 advocates
- ENTERPRISE tier: unlimited advocates
