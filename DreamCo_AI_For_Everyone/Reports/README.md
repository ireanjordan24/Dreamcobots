# DreamCo AI For Everyone — Reports

## Overview
This directory contains generated reports from the AI Enablement Hub and
other monitoring systems.

---

## Report Types

| Type | Frequency | Generated By |
|------|-----------|-------------|
| Monthly Adoption Report | Monthly | AI Enablement Hub workflow |
| Revenue Report | Monthly | Revenue Engine Bot |
| Productivity Report | Quarterly | AI Enablement Hub workflow |
| Security Audit | Quarterly | Manual + automated |

---

## Generating Reports

### Monthly Summary Report
```
Actions → AI Enablement Hub → Run workflow → report_type=summary
```

### Monthly Revenue Report
```
Actions → AI Enablement Hub → Run workflow → report_type=revenue
```

### Maturity Assessment
```
Actions → AI Enablement Hub → Run workflow → report_type=maturity
```

---

## Report Archive
Reports are committed to this directory by the AI Enablement Hub workflow.
File naming convention: `{type}_YYYY_MM.md`

Example: `summary_2025_01.md`, `revenue_2025_01.md`
