## 🏗️ CORE SYSTEMS {#core-systems}

### Framework
- **`framework/global_ai_sources_flow.py`** — The **mandatory 8-stage AI pipeline** every bot must implement:
  ```
  GLOBAL AI SOURCES → Data Ingestion → Learning Classifier → Sandbox Test Lab
    → Performance Analytics → Hybrid Evolution Engine → Deployment Engine
    → Profit & Market Intelligence → Governance + Security
  ```
- **`framework/retraining_optimizer.py`** — Drift detection and automated retraining strategy selector

### Core Infrastructure
- **`core/base_bot.py`** — BaseBot class with `run(task)`, `monetize(amount, source)`, `report()` methods. Foundation for all bots.
- **`bots/buddy_orchestrator/buddy_orchestrator.py`** — Master orchestrator: `run_all_async()`, `analytics()`, `white_label_bot()`, governance KPIs, advocacy KPIs, onboarding metrics, AI fluency status
- **`global_learning_system/learning_loop.py`** — `GlobalLearningLoop`: persists decisions to SQLite, extracts successful strategies, gates retraining
- **`dreamco/revenue_orchestrator.py`** — `RevenueOrchestrator`: runs all profitable bots, aggregates revenue, tracks status
- **`config/config_manager.py`** — Master config manager for profit targets, thresholds, and system settings

### AI & Intelligence Systems
- **`bots/ai_brain/`** — `DecisionEngine`: classifies situations (low leads → scale_leads, low revenue → increase_outreach, high revenue → scale_system, optimize for medium)
- **`bots/deep_learning_system/`** — `DeepLearningCoordinator` with 3 engines: `APIScraperEngine` (top-1000 APIs/category), `CompetitorAnalysisEngine` (pricing/sentiment), `SandboxTestingEngine`
- **`bots/global_ai_learning_matrix/`** — Cross-bot shared learning matrix
- **`bots/global_bot_network/bot_library.py`** — Central registry of all bots

### SaaS & Auth
- **`saas/auth/api_key_registry.py`** — `ClientApiKeyRegistry`: 5 key categories (read_only, full_access, bot_runner, webhook, analytics), keys prefixed `dc_<tier>_<hex32>`
- **`api/app.py`** — REST API: `POST/GET /api/tokens`, `DELETE /api/tokens/<id>`, `GET /api/tokens/usage`, OAuth for Google/GitHub
- **`stripe/subscription_handler.py`** — Stripe subscription lifecycle management
- **`bots/ai_enablement_hub/ai_enablement_hub.py`** — FREE/PRO/ENTERPRISE tiers; 5 pillars: AdvocatesProgram, PoliciesGuardrails, LearningDevelopment, DataMetrics, CommunityOfPractice

### Domain Management
- **`bots/business_launch_pad/domain_manager.py`** — 4 domain statuses: owned/listed/sold/flipped; register, valuate, flip, portfolio_summary
- **`DreamCo/bots/domainBot.js`** — JavaScript domain bot
- **`DreamCo/firebase/schemas.js`** — Firestore schema for domains collection
- **`DreamCo/server.js`** — Express server with domain endpoints (14–19)

### Frontend / Dashboards
- **`dreamco-control-tower/`** — React Control Tower (7 tabs: Bot Overview, Repo Activity, Bot Deployment, Analytics, Actions, Marketplace, Learning)
- **`frontend/screens/developer_settings_screen.py`** — API key management + OAuth provider buttons
- **`frontend/screens/domain_manager_screen.py`** — Domain portfolio UI

### Specialty Tool Libraries
- **`automation-tools/`** — Workplace Audit Tool (5S methodology), Color Palette Generator, Smart Meeting Scheduler
